# README

## サービス名
筋肉雨予報

## 概要
筋トレ好きのための降水確率を通知するLINE BOTです。

1.定時通知
東京の降水確率が30%以上の日の毎朝6時に降水確率を通知

2.コメント応答
「今日」や「明日」、「あいさつ」などいくつかの種類のコメントに対し応答

3.「定時通知」および「コメント応答」時のいずれにも、筋トレのモチベーションを高めるメッセージを付加
